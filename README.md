Redmine Additional Filters plugin
======================

This Redmine plugin adds some useful filters and columns.

Search issues by:
- Notes
- All text fields (Subject, Description and Notes)
- Number of notes
- ...

Display new columns as:
- First assignment date
- Resolved date  
- Author mail address


## Requirements

    ruby >= 2.1.0
